# Retail Sales Performance & Customer Analytics Dashboard

## 📊 Project Overview

This project is an end-to-end retail analytics solution built using **Python, MySQL, and Power BI**.

The project covers data generation, database loading, data modeling, and interactive dashboard development to analyze sales performance, customer behavior, product insights, and regional performance.

---

## 🛠 Tools Used

- Power BI
- MySQL
- Python (Pandas, Faker)
- Excel
- DAX
- Data Modeling

---

## 🔄 Project Workflow

**Python → Excel → MySQL → Power BI**

- Generated realistic retail datasets using Python.
- Stored data in Excel files.
- Uploaded datasets into MySQL using Python and SQLAlchemy.
- Connected MySQL database with Power BI.
- Built a Star Schema data model and created interactive dashboards.
